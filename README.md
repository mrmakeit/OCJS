# OCJS

## Setup

Setup dev environment
`./gradlew setupDevWorkspace`

If you need javascript debugging, add org.eclipse.wst.jsdt.debug.* files from Eclipse wst

## Building

`./gradlew build`

## Running

`./gradlew runClient`
